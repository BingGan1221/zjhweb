# 评论分析系统

## 功能特点
- 多文件合并分析
- 数据来源分布统计
- 词频分析和词云图
- 差评、建议、负面情绪分析
- 自定义停用词管理

## 使用说明
1. 上传Excel文件（支持多选）
2. 使用筛选条件过滤数据
3. 查看不同维度的分析结果
4. 管理自定义停用词
